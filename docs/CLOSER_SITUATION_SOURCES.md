# MLB Closer Situation Monitor (NSV Category)

## 1. Overview
Net Saves + Holds (NSV) is the gold standard for evaluating high-leverage relief value in 2026. Monitoring "Closer Situations" requires a combination of real-time usage alerts and "leverage ladder" depth charts.

## 2. Industry Standard Monitoring Sources
| Source | URL | Focus | Best Use Case |
| :--- | :--- | :--- | :--- |
| **Closer Monkey** | [closermonkey.com](http://www.closermonkey.com) | Real-time closer changes. | Instant bullpen role shift alerts. |
| **Pitcher List** | [pitcherlist.com/closer-encounters](https://www.pitcherlist.com/closer-encounters) | Daily bullpen deep-dive articles. | Identifying "Vulture Save" opportunities. |
| **RosterResource** | [fangraphs.com/roster-resource/bullpen-depth-charts](https://www.fangraphs.com/roster-resource/bullpen-depth-charts) | Detailed bullpen hierarchy. | Long-term stash candidates for Holds. |

## 3. NSV Category Strategies
*   **Holds Tracking:** Focus on "Middle Relief (MR)" pitchers with elite Stuff+ (115+) who are not yet in the closer role.
*   **Vulture Saves:** Monitor closer "Fatigue" via the **RosterResource Bullpen Usage Chart**. If a closer has pitched 3 days in a row, the primary set-up man is a high-EV daily pick.

## 4. Automation Focus
*   **Closer Monkey Twitter/X Scraper:** Capture "Closer Change" alerts for instant free-agent acquisition.
*   **NSV Rankings:** Pull from FanGraphs or RotoBaller to update season-long projections.
