**Science behind weather scoring:**
- Every 10°F ≈ 3-4% more distance on fly balls
- 10mph wind out = 15-20% more HRs
- 10mph wind in = 15-20% fewer HRs
- Elevation (Coors Field) = ~15% boost at altitude

**SmartLineupSelector updates:**
- Weather boost in scoring: `(hitter_friendly_score - 5.0) * 0.5`
- HR weather boost for power hitters: `(hr_factor - 1.0) * proj_hr * 10`
- Weather warnings for high-risk games

### Pitcher Start Detection Fix
**File:** `backend/fantasy_baseball/daily_lineup_optimizer.py`

**Bug:** Shota Imanaga showing as START when he doesn't pitch today

**Root cause:** `flag_pitcher_starts()` only checked if pitcher's TEAM had a game, not if the specific pitcher was the probable starter.

**Fix:**
```python
# Before (wrong):
has_start = team in team_odds  # Any Cubs pitcher shows START

# After (correct):
probable = _fetch_probable_pitchers_for_date()  # MLB Stats API
is_probable = _is_probable_starter(name, team, probable)  # Fuzzy match
has_start = has_game and is_probable  # Must be actual starter
```

**New methods:**
- `_fetch_probable_pitchers_for_date()`: Queries MLB Stats API for probable pitchers
- `_is_probable_starter()`: Fuzzy name matching to verify specific SP
