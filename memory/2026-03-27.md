### dd6ce84 - Add Weather Integration for Baseball Decisions
**File:** `backend/fantasy_baseball/weather_fetcher.py`

Weather is now fully integrated into lineup decisions:

**GameWeather dataclass:**
- Temperature, wind (speed/direction), humidity
- Precipitation chance, conditions
- `hitter_friendly_score`: 0-10 rating
- `hr_factor`: Multiplier (1.0 = neutral, 1.25 = 25% more HRs)
- `game_risk`: low/medium/high/postponement_risk

**WeatherFetcher:**
- All 30 MLB venues mapped to cities
- Stadium profiles (dome, retractable, elevation)
- OpenWeatherMap API integration
- Fallback to seasonal estimates if no API key
- 30-minute cache

**Science behind weather scoring:**
- Every 10°F ≈ 3-4% more distance on fly balls
- 10mph wind out = 15-20% more HRs
- 10mph wind in = 15-20% fewer HRs
- Elevation (Coors Field) = ~15% boost at altitude

**SmartLineupSelector updates:**
- Weather boost in scoring: `(hitter_friendly_score - 5.0) * 0.5`
- HR weather boost for power hitters: `(hr_factor - 1.0) * proj_hr * 10`
- Weather warnings for high-risk games

**Briefing factors now include:**
- "☀️ Hitter weather (82°F)"
- "💨 15mph wind OUT"
- "⚾ HR boost (1.25x)"
